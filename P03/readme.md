# P03: Kanban del projecte

## Breu descripció
Com en projectes anteriors, la metodologia **Kanban** és vital pel bon funcionament del projecte. Per aquest motiu:

- A partir del **product backlog** proporcionat, heu de fer el seguiment de les vostres realitzacions del projecte.
- Cada setmana es farà una **revisió de l’estat del Planner**, així que és molt important **no deixar-lo per última hora**.

---

## Product Backlog
- Es proporcionarà el **llistat de tasques i funcionalitats** que cal dur a terme durant el projecte.
- Cada tasca ha de tenir un **estat clar**: `Per fer`, `En procés`, `Finalitzat`.

## Recomanació
- Actualitzar el Kanban de manera **regular**.
- Assignar **responsabilitats clares** a cada membre de l’equip.
- Utilitzar etiquetes o colors per diferenciar tipus de tasques o prioritat.
